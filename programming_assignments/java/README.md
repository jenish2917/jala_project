# Java Basics Assignment

This project contains a Java program that demonstrates basic Java programming concepts as part of the JALA Academy assignments.

## Features

- Variables and data types
- Operators (arithmetic, comparison)
- Loops (for, while)
- Arrays
- Static methods
- String operations
- Inheritance
- Access modifiers
- Abstract classes
- Exception handling
- Collections
- Constructors

## Files

- `src/JavaBasicsAssignment.java` - Main Java program

## How to Run

1. Make sure you have Java JDK installed
2. Navigate to the src directory: `cd src`
3. Compile the program: `javac JavaBasicsAssignment.java`
4. Run the program: `java JavaBasicsAssignment`

## Assignment Coverage

This implementation covers basic Java concepts in a simple, easy-to-understand manner with clear explanations.
