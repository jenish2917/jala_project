# C# Basics Assignment

This project contains a C# program that demonstrates basic C# programming concepts as part of the JALA Academy assignments.

## Features

- Variables and data types
- Operators (arithmetic, comparison)
- Loops (for, while)
- Arrays
- Static methods
- String operations
- Inheritance
- Access modifiers
- Abstract classes
- Exception handling
- Collections
- Constructors

## Files

- `src/CSharpBasicsAssignment.cs` - Main C# program

## How to Run

1. Make sure you have .NET SDK installed
2. Navigate to the src directory: `cd src`
3. Compile and run the program: `dotnet run` (if using .NET project) or compile with `csc CSharpBasicsAssignment.cs` and run with `CSharpBasicsAssignment.exe`

## Assignment Coverage

This implementation covers basic C# concepts in a simple, easy-to-understand manner with clear explanations.
