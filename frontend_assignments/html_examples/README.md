# JALA Academy Frontend Assignment

This project demonstrates HTML5, CSS, and JavaScript concepts as part of the JALA Academy frontend assignments.

## Features

### HTML5 Semantic Elements
- `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- Proper document structure and accessibility
- Form elements with validation

### CSS Features
- Flexbox and Grid layouts
- Responsive design with media queries
- CSS animations and transitions
- Gradient backgrounds
- Box shadows and border-radius
- Pseudo-elements and pseudo-classes

### JavaScript Features
- DOM manipulation
- Event handling (click, focus, blur, submit)
- Form validation
- Local storage
- Array and object methods
- Promises and async/await
- ES6+ features (classes, arrow functions, etc.)
- Dynamic content creation

## Files

- `index.html` - Main HTML structure with semantic elements
- `styles.css` - All styling including responsive design
- `script.js` - JavaScript functionality

## How to Run

1. Open `index.html` in a web browser
2. The page includes interactive elements like:
   - Form with validation
   - Clickable service cards
   - Smooth scrolling navigation
   - Dynamic content loading
   - Real-time input validation

This implementation covers:
- HTML5 basics and semantic elements
