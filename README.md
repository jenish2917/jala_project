## Assignment Details

### Cyber Security Assignments

#### Cyber Security Fundamentals
1. Encrypt "HELLO" using Caesar cipher with shift of 4, then decrypt "Jg qh iwtg" using reverse shift
2. Generate RSA key pair using OpenSSL and export in PEM format

#### Operating System Security
1. Create Linux user accounts with specific file permissions
2. Disable SSH root login and verify

#### Network Security & VPNs
1. Set up OpenVPN and verify IP change
2. Use nmap to scan ports and identify services

#### Web Application Security
1. Perform SQL injection on vulnerable application
2. Perform XSS attack with JavaScript alert

#### Ethical Hacking & Penetration Testing
1. Use whois and nslookup for domain information
2. Use Metasploit Framework to exploit vulnerabilities

#### Malware Analysis & Reverse Engineering
1. Use strings command to extract text from malware
2. Set up sandbox and monitor malware behavior

#### Incident Response & Digital Forensics
1. Simulate security breach and create audit log
2. Analyze system logs for suspicious activity

#### Cloud & IoT Security
1. Set up VM in cloud with firewall configuration
2. Connect IoT devices and capture network traffic

### Data Science Assignments

#### Basic Array Operations
1. Create 3x3 array with random integers, calculate sum, mean, std dev
2. Create 1D array and compute cumulative sum
3. Generate arrays and perform element-wise operations
4. Create 4x4 identity matrix
5. Use broadcasting to divide array elements

#### Array Manipulation
1. Reshape 1D array to 3x4 matrix
2. Flatten 3x3 matrix to 1D array
3. Stack matrices horizontally and vertically
4. Concatenate arrays along new axis
5. Transpose and reshape matrix

#### Indexing and Slicing
1. Extract elements with specific step
2. Extract sub-matrix from 5x5 matrix
3. Replace elements greater than 10
4. Use fancy indexing to select elements
5. Reverse array using slicing

#### Broadcasting
1. Add 1x3 array to each row of 3x3 matrix
2. Multiply array by scalar using broadcasting
3. Subtract column vector from matrix
4. Add scalar to 3D array
5. Add arrays of shapes (4,1) and (1,5)

#### Vectorized Operations
1. Compute square root of each element
2. Calculate element-wise absolute difference
3. Perform element-wise power operation
4. Calculate dot product of two arrays
5. Implement vectorized function for complex calculations
